# Requirements Document: Course Live Sessions Display

## 1. Functional Requirements

### 1.1 Live Sessions Display
The system shall display recently scheduled live sessions at the top of the course content page for students.

**Acceptance Criteria:**
- Live sessions banner appears above the course video player
- Banner is visible immediately when student navigates to course content page
- Banner displays maximum 3 sessions
- Banner shows session topic, instructor name, start time, duration, and status
