# Partner Logos Directory

This directory contains logos for B2B partner companies displayed on the platform.

## Add Your Logos Here

Place your company logo files in this directory with these names:

- `techcorp.png`
- `global-innovations.png`
- `digital-dynamics.png`
- `enterprise-systems.png`
- `cloudtech.png`
- `dataflow.png`
- `nextgen.png`
- `smart-solutions.png`
- `innovate-labs.png`

## Logo Specifications

- **Format**: PNG with transparent background (or SVG)
- **Size**: 200x80px (or 400x160px for retina displays)
- **File Size**: < 100KB
- **Naming**: Use kebab-case matching the database entries

## Quick Start

1. Add your logo files to this directory
2. Ensure file names match the paths in the database
3. Refresh your admin panel to see the logos
